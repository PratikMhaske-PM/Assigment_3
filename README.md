# 📘 Assignment 3 – Logic Improvement (JavaScript / TypeScript)

## 📌 Overview

This assignment focuses on improving programming logic using JavaScript/TypeScript. It covers fundamental concepts such as loops, conditionals, arrays, strings, and basic algorithms.

---

## 🛠️ Technologies Used

* JavaScript / TypeScript
* VS Code (or any code editor)
* Node.js (for execution)

---

## 📂 Questions Covered

### 1. Even and Odd Numbers

* Created two separate arrays for even and odd numbers between 0 to 100.

### 2. Star Patterns

* Printed:

  * Pyramid pattern
  * Increasing triangle
  * Decreasing triangle

### 3. Prime Numbers

* Identified prime numbers within a given range and stored them in arrays.

### 4. Palindrome Number

* Checked whether a given number is a palindrome.

### 5. Factorial

* Calculated factorial of a number using loops.

### 6. ASCII Value

* Converted a character into its ASCII value.

### 7. Matrix Operations

* Performed addition and subtraction of two matrices.

### 8. Maximum Frequency Character

* Found the character with the highest frequency in a string.

### 9. Object Sorting

* Sorted an object based on its keys.

### 10. Linear Search

* Implemented linear search to find an element in an array.

### 11. Insertion Sort

* Sorted an array using insertion sort algorithm.

### 12. Reverse Linked List

* Implemented logic to reverse a linked list.

## 🧠 Key Concepts Learned

* Looping (for, while)
* Conditional statements
* Array manipulation
* String handling
* Algorithm design
* Problem-solving approach

## 🚀 How to Run

1. Install Node.js
2. Save file as `.ts` or `.js`
3. Run using:

```bash
node filename.js
```

(For TypeScript, compile first using `tsc filename.ts`)
